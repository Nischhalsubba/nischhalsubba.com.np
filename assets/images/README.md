## Rules
- Prefer local SVG or optimized WebP assets.
- Use descriptive alt text in HTML.
- Avoid fake screenshots. If a project is conceptual or in progress, use clearly illustrative graphics.
- Replace SVG mockups with real Figma exports only when the image is public-safe and approved.
